Procedural memory inicializada com 11 nodes (dev/patterns + sheldon/architecture-decisions). Loading lazy via _index.json.

2026-05-07
Pentester rounds 4-7 — hardening completo security

execSync → spawn com array args, três camadas de defesa contra injection (LLM01.2), CI endurecido, installer protegido contra downgrade silencioso.

2026-05-11
Living Memory Phases 0-5 memory

Reflect engine + autonomy v1.1 + doctor + docs pt-BR + /discover em todas as harnesses. Bootstrap cache de 4 arquivos.

2026-05-11
Deyvin-density — framework MICRO-aware fix

Handoff contracts cientes de MICRO, transição automática entre features no workflow. Inception fix que destrava MICRO dentro de MEDIUM.

2026-05-14
Active Learning Loop + Subtask Scout learning

Telemetry foundation → FTS5 search → archive com evolution log → doctor curation → distillation hook. Mais 9 fixes cross-platform Windows.

2026-05-15
Security Round 7 — SF-21 a SF-26 security

Sanitização de memory-context writers, brain-query fail-closed, runtime-permissions com deny rules, delegation-handoff carve-out de Gate D.

2026-05-16
Release v1.9.0 + Neo upgrade + Handoff persistence release

Installer com merge JSON aditivo (não sobrescreve customizações), @neo enxerga Living Memory + harness + dossiers + brains, diagnóstico persiste antes de /clear.

2026-05-17
Lay-user agent mode + dev-state self-producer ux

Profile creator por padrão com 1 pergunta por turno, jargão traduzido, opção "Pausar / quero pensar". @analyst e @product passam a escrever dev-state.md nativamente.

📌 Self-eating feature

Dev-state auto-produzido

@analyst e @product escrevem dev-state.md nativamente no fim de cada sessão via dev:state:write. Tokens canônicos com warn-and-skip. Parser truncation fix bonus.

aioson dev:state:write . --feature=checkout \ --next="Phase 2" --context=prd,requirements
Próximos passos

O que vem depois.

Pontas soltas confirmadas durante o sprint e priorizadas para as próximas releases.

  • Autonomy v1.1 cross-project — promover o contrato de 3 tiers para projetos consumidores, com telemetry de adoção em SQLite.
  • Harness-isolation — sandbox real por tier (deferred do living-memory). Foi a única decisão adiada do sprint.
  • Kernel dev.md < 15 KB — o kernel cresceu durante o sprint; precisa trim antes do v1.10.
  • Multi-agent scout — extensão do deyvin-subtask-scout quando houver dados de uso V1.
  • Cobertura docs nos 4 idiomas — AC-ALL-604 deferred do active-learning-loop.
