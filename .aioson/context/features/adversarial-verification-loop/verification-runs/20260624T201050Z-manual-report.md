## Before And Now

Before Slice 3, schema validation lived inside parser modules and agent prompts did not know when to prepare or consume the verification artifacts. - `--promote` and telemetry are deferred, so latest-report management is still manual.
- Cross-tool execution is still absent by design; no Codex, Claude, or opencode runner was invoked.
